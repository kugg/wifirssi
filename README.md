wifirssi
========

Wifi Received Signal Strength Indicator

Sample program to detect signal failures in Wifi.
GNU/GPLv3 Christoffer Jerkeby 2014

Dependencies:
------------

 * pythonwifi
 * matplotlib (python builtin)
 * linux

 Installation:

$ pip install pythonwifi

Usage:
------

 * Connect to wifi
 * python ./wifirssi.py
